# 👋 سلام، من محمد هستم

**مهندس نرم‌افزار | توسعه‌دهنده Backend | متخصص هوش مصنوعی و بلاکچین**

---

## 🧭 درباره من

مهندس نرم‌افزار با تجربه در طراحی و ساخت سیستم‌های Backend مقیاس‌پذیر، یکپارچه‌سازی هوش مصنوعی، و توسعه راه‌حل‌های بلاکچین. تخصصم در معماری APIهای REST و GraphQL، مدیریت داده‌های توزیع‌شده، و خودکارسازی pipeline های CI/CD است.

برای من، نوشتن کد تمیز، قابل‌تست و قابل‌نگهداری یک اصل پایه است — نه یک گزینه. به ساخت محصولاتی که تأثیر واقعی دارند اعتقاد دارم، از ابزارهای بهره‌وری هوشمند گرفته تا سامانه‌های هوشمند کشاورزی و راه‌حل‌های Web3.

مشارکت فعال در اکوسیستم متن‌باز دارم و با **+۷۵ ریپوزیتوری عمومی**، پیوسته در حال یادگیری، ساختن و به اشتراک‌گذاری هستم.
