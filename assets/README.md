این پوشهٔ `assets/` برای تصاویر و GIFهای پروژه‌ها است.

راهنما:
- فایل‌های بصری پروژه‌ها (screenshots, demo.gif) را در این پوشه قرار بده.
- در README از مسیر نسبی مثل `./assets/demo.gif` استفاده کن.

مثال:
- assets/demo.gif  -- GIF یا ویدیوی کوتاه از پروژه
- assets/screenshot-1.png -- اسکرین‌شات

(در حال حاضر این پوشه خالی است؛ لطفاً فایل‌های رسانه‌ای را آپلود کن تا من README را با لینک‌های درست به‌روز کنم.)
